Úvod

Tento dokument analyzuje data ze sezóny NBA a zaměřuje se na rozdíly ve skóre mezi domácími a hostujícími týmy, celkový počet výher týmu a statistiky jednotlivých hráčů.

Načtení dat

Zde využíváme funkci dbGetQuery pro provedení dotazu a načtení výsledku do R, a následně s nimi pracujeme již v R. Druhou možností je použití dbReadTable, která načte do R celou tabulku. Zvolili jsme první možnost, protože je jednodušší poté změnit, které sloupce nám například do R stačí načíst, když nechceme všechny. Třetí možností je knihovna dbplyr a příkaz tbl – data se nenačítají celá do R paměti, ale místo toho dbplyr generuje SQL dotazy, které se provádějí přímo na serveru. Při menších datech nevzniká s našim postupem problém. Při větších datech může být výhodnější používat dbplyr, na druhou stranu však automatické překládání na dotaz do SQL může přinášet při složitějších konstrukcích problémy.

Histogram rozdílů skóre

Nejprve získáme rozdíl skóre mezi domácím a hostujícím týmem. Použijeme absolutní hodnotu, protože nám nezáleží na tom, kdo vyhrál, pouze na velikosti rozdílu ve skóre. Rozdíly jsou rozděleny do kategorií, abychom mohli lépe porozumět tomu, jak těsné byly zápasy v jednotlivých rozmezích.

# Kategorie rozdílů skore po dvou bodech a nadkategorie po 6 bodech
nba_zapasy <- nba_zapasy %>%
  mutate(diference_skore = abs(domaci_skore - hoste_skore)) %>%
  filter(diference_skore > 0) %>%
  mutate(
    kategorie = case_when(
      diference_skore <= 2   ~ "1-2",  diference_skore <= 4   ~ "3-4",
      diference_skore <= 6   ~ "5-6",  diference_skore <= 8   ~ "7-8",
      diference_skore <= 10  ~ "9-10", diference_skore <= 12  ~ "11-12",
      diference_skore <= 14  ~ "13-14",diference_skore <= 16  ~ "15-16",
      diference_skore <= 18  ~ "17-18",diference_skore <= 20  ~ "19-20",
      diference_skore <= 22  ~ "21-22",diference_skore <= 24  ~ "23-24",
      diference_skore <= 26  ~ "25-26",diference_skore <= 28  ~ "27-28",
      diference_skore <= 30  ~ "29-30",TRUE                   ~ "31+"
    ),
    nadkategorie = case_when(
      diference_skore <= 6   ~ "1-6 bodů",  diference_skore <= 12  ~ "7-12 bodů",
      diference_skore <= 18  ~ "13-18 bodů",diference_skore <= 24  ~ "19-24 bodů",
      diference_skore <= 30  ~ "25-30 bodů",TRUE                   ~ "31+ bodů"
    )
  )


# Pořadí kategorií (pro grafy a legendy) neboť kategroie a nadkategorie jsou vytvořené jako textové řetězce a R je defaultně v grafech řadí alfabeticky

Heatmapa

Heatmapa skóre domácích týmů podle měsíců. Heatmapa vizualizuje průměrné skóre jednotlivých domácích týmů v průběhu sezóny. Umožňuje rychle odhalit výkonnostní výkyvy mezi měsíci.

# Spojení s tabulkou nba_tymy a výpočet průměrů
heatmapa_data <- nba_zapasy %>%
  mutate(mesic = format(as.Date(datum), "%Y-%m")) %>%
  group_by(id_tym_domaci, mesic) %>%
  summarise(prumer_bodu = mean(domaci_skore), .groups = "drop") %>%
  left_join(nba_tymy, by = c("id_tym_domaci" = "tym_id"))  # spojení pro název týmu

# Vykreslení heatmapy
ggplot(heatmapa_data, aes(x = mesic, y = nazev, fill = prumer_bodu)) +
  geom_tile(color = "white") +
  scale_fill_gradient(low = "lightyellow", high = "red") +
  labs(
    title = "Heatmapa skóre domácích týmů podle měsíců",
    x = "Měsíc", y = "Domácí tým", fill = "Průměr bodů"
  ) +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1))

Vývoj průměrných bodů v zápasech po týdnech v čase

Nyní se podíváme na průměrné body v zápasech po jednotlivých hracích týdnech. Budeme pozorovat jak se v čase mění ofenzivní nasazení týmů v lize a pokusíme se výsledný graf proložit spojitou křivkou.

# Vytvoření týdenních dat
skore_v_case <- nba_zapasy %>%
  mutate(tyden = floor_date(as.Date(datum), "week")) %>%
  group_by(tyden) %>%
  summarise(prumer_skore = mean(domaci_skore + hoste_skore))

# GRAF
ggplot(skore_v_case, aes(x = tyden, y = prumer_skore)) +
  geom_line(color = "steelblue", size = 1.2) +
  
  # Proložení LOESS
  geom_smooth(aes(color = "LOESS"), method = "loess", se = FALSE, size = 1.2) +
  
  # Proložení polynomem 3. stupně
  geom_smooth(aes(color = "Polynom 3. stupně"),
              method = "lm", formula = y ~ poly(x, 3),
              se = FALSE, size = 1.2, linetype = "dashed") +
  
  scale_color_manual(values = c("LOESS" = "darkred", "Polynom 3. stupně" = "forestgreen")) +
  
  labs(
    title = "Vývoj průměrného skóre v NBA během sezóny (týdně)",
    x = "Týden", y = "Průměrné skóre", color = "Metoda proložení"
  ) +
  
  theme_minimal() +
  theme(
    legend.position = "right",
    axis.text.x = element_text(angle = 45, hjust = 1)
  )

Koláčový graf výher týmů

Nyní zobrazíme koláčové grafy poddílů výher týmů ve všech 6 divizích. Zde se vyskytoval problém se spojováním mnoha záznamů a celý R Markdown dokument se následně renderoval déle, než bylo vhodné. V tomto případě však při prozkoumání nebyl problém z pamětí v R. Je však třeba brát na paměť, že náš způsob někdy může být kamenem úrazu.

# Spočítání výher pro týmy v zápasech
nba_vitezstvi <- nba_zapasy %>%
  mutate(vitez_id = ifelse(domaci_skore > hoste_skore, id_tym_domaci, id_tym_hoste)) %>%
  count(vitez_id, name = "vitezstvi")

# Spojení informací o týmech (divize, konference)
info_tymy <- nba_tymy %>%
  left_join(nba_divize, by = "divize_id") %>%
  left_join(nba_konference, by = "konference_id") %>%
  select(tym_id = tym_id, tym = zkratka.x, divize = nazev.y, konference = zkratka)

# Spojení s výhrami
nba_prehled <- info_tymy %>%
  left_join(nba_vitezstvi, by = c("tym_id" = "vitez_id")) %>%
  mutate(vitezstvi = replace_na(vitezstvi, 0)) %>%
  select(tym_id, tym, divize, konference, vitezstvi) %>%
  arrange(divize, desc(vitezstvi))  # Seřazení podle divize a výher

# Spočítání procent výher v rámci divize
nba_prehled_procent <- nba_prehled %>%
  group_by(divize) %>%
  mutate(podil = vitezstvi / sum(vitezstvi)) %>%
  ungroup()

Interaktivní zobrazení statistik hráčů

Nyní se zaměříme na individuální výkony hráčů zobrazíme si tabulkově nejlepší hráče dle bodů v lize. Dále spočítáme ligové průměry některých statistik jako jsou body atp. Na závěr uděláme interaktivní vizualizaci, kdy si čtenář bude moct vybrat hráče, zobrazit jeho statistiky a porovnat je s ligovými průměry. Zde opět upozorňujeme na zdroj a formát dat, autor více rozebere na místě při prezentaci projektu.

# Získání názvů týmů, za které hráč hrál
tymy_hracu <- nba_sezona_hraci %>%
  left_join(nba_tymy, by = "tym_id") %>%
  group_by(hrac_id) %>%
  summarise(
    tymy = paste(unique(zkratka), collapse = ", ")
  )


# Spojení statistik a týmů, KVULI forme dat a ulozeni v databazy nelze například použít mean(více se o tomto rozpovídám na prezentaci)
statistiky_hracu <- nba_sezona_hraci %>%
  group_by(hrac_id) %>%
  summarise(
    pocet_zapasu = sum(G), 
    pocet_zapasu_z5 = sum(GS),
    celkem_body = sum(PTS * G, na.rm = TRUE),  
    prumer_body_na_zapas = sum(PTS * G, na.rm = TRUE) / sum(G, na.rm = TRUE),  
    celkem_doskoky = sum(TRB * G, na.rm = TRUE),  
    prumer_doskoky_na_zapas = sum(TRB * G, na.rm = TRUE) / sum(G, na.rm = TRUE),
    celkem_asistence = sum(AST * G, na.rm = TRUE),  
    prumer_asistence_na_zapas = sum(AST * G, na.rm = TRUE) / sum(G, na.rm = TRUE),
    efektivita_strelby = round(sum(eFG * G, na.rm = TRUE) / sum(G, na.rm = TRUE), 2),  
    prumer_minut = round(sum(MP * G, na.rm = TRUE) / sum(G, na.rm = TRUE), 2)
  ) %>%
  left_join(nba_hraci, by = "hrac_id") %>%
  left_join(tymy_hracu, by = "hrac_id") %>%
  arrange(desc(prumer_body_na_zapas))
